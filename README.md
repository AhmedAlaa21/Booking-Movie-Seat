# Booking-Movie-Seat

Vox Cinema Lookalike website, in which the user can book the available seats in theatre.

## Live Demo
[Movie Seat Booking](https://booking-movie-seat.vercel.app/) - book your seat

## Specifications

- Designed with HTML and CSS
- Functionality with JavaScript
- store data using local Storage and Json stringfy
